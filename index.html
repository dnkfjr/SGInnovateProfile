Programs that I Created

OFFVIZ

It is a purpose made post-processing tool developed with VBA for Excel. The program completely automates the post-processing of text-based computational output from a specialized submarine pipe-laying analysis software OFFPIPE Ver2.0 and provides visualization of key results for evaluation by the pipelay analysis engineer

AutoLAY

Developed in Python 2, the program interfaces with a state-of-the-art software used in oil-and-gas industry for pipe-laying analysis.

The program allows fast analysis to be carried out from computer terminal instead of user having to open and manually interact with the software via GUI. Iteration procedures are built into the program to allow for fast preliminary estimation before starting a detailed analysis. Extraction of analysis results are done automatically which is beneficial in the preliminary stage of analysis

Seabed Data Check

A simple program developed in IPython Notebook / Python 2 to evaluate the completeness of client-provided seabed geophysical survey data for a planned three-130km-long submarine pipelines in water depths from 0 to 60m below. The digitized survey data has resolution of 0.5m to 1m covering large survey areas with data file size ranging from 100MBs to GBs.

The program takes in the seabed depth readings and mapped them to grayscale value and draw them on an image file. Missing survey data are easily identified visually by observing areas of planned pipeline routes without drawn data.
